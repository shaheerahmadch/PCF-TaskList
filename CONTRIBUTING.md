# Contributing

Thank you for your interest in contributing to the **TaskList** component! We welcome your help to improve this project.

## How to Contribute

1.  **Fork the repository**: Start by forking this repository to your own GitHub account.
2.  **Create a Branch**: Create a new branch for your feature or bug fix.
    ```bash
    git checkout -b feature/your-feature-name
    ```
3.  **Make Changes**: Implement your changes, ensuring you follow the existing code style and conventions.
4.  **Test**: Verified your changes work as expected in a test environment.
5.  **Commit**: Commit your changes with clear and descriptive commit messages.
    ```bash
    git commit -m "Add feature: ..."
    ```
6.  **Push**: Push your branch to your forked repository.
    ```bash
    git push origin feature/your-feature-name
    ```
7.  **Pull Request**: Open a Pull Request (PR) against the `main` branch of this repository.

## Reporting Issues

If you encounter any bugs or have feature requests, please open an issue in the [Issues](https://github.com/shaheerahmadch/Canvasify-Components/issues) section of the repository. Provide as much detail as possible to help us understand and resolve the problem.

## Code of Conduct

Please be respectful and considerate of others when contributing to this project.
